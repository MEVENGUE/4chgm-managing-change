# NEXORA OS — Brief produit v1.0

## Positionnement
NEXORA OS est l’interface de pilotage de la transformation pour décideurs et équipes delivery. Il doit donner l’impression d’un OS stratégique : propre, premium, animé, mais sans gaming ni surcharge.

## Direction artistique
- Dark premium : fonds très sombres, surfaces glass, typographie serif pour les titres, sans-serif pour les actions, glow discret, espace respirant.
- Light premium : tons crème/blanc cassé, ombres douces, bordures subtiles, même hiérarchie typographique.

## Palette cible
- Primary : slate/gray + cyan/indigo/violet selon le contexte.
- Accents fonctionnels : emerald pour succès, amber pour risque, rose pour critique.
- Surfaces : rgba(255,255,255,0.06) ou rgba(0,0,0,0.02) selon le thème.

## Principes UI
- Premium minimalisme, pas cheap.
- Hiérarchie claire : sections, métriques, insights, actions.
- Layout dashboard : sidebar + topbar + zone principale.
- Composants réutilisables : panels glass, cartes KPI, chips, indicateurs de statut.
